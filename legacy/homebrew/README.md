# Homebrew Tap for Ifá-Lang

This is the Homebrew tap for [Ifá-Lang](https://github.com/AAEO04/ifa-lang) - The Yoruba Programming Language.

## Installation

```bash
# Add the tap
brew tap AAEO04/ifa-lang

# Install Ifá-Lang
brew install ifa-lang
```

## Usage

```bash
ifa --help
ifa run hello.ifa
ifa repl
```

## Update

```bash
brew upgrade ifa-lang
```

## Uninstall

```bash
brew uninstall ifa-lang
brew untap AAEO04/ifa-lang
```
