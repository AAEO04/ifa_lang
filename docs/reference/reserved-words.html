Reserved Words

Bilingual Keywords and Language Primitives

Overview

Ifá-Lang supports bilingual keywords with both Yoruba and English variants. All reserved words have cultural significance and reflect Yoruba philosophical concepts while maintaining accessibility for English speakers.

Variable and Function Declarations

ayanmo / let

Meaning: Destiny / to have

Usage: Variable declaration

ayanmo x = 5;
let name = "Alice";

ese / fn

Meaning: Verse / function

Usage: Function definition

ese greet(name) { ... }
fn calculate(x, y) { ... }

odu / class

Meaning: Odù (sacred pattern) / class

Usage: Type/structure definition

odu Person { ... }
class Database { ... }

const / loruko

Meaning: Constant / name

Usage: Constant declaration

const PI = 3.14159;
loruko VERSION = "1.3.0";

public

Meaning: Public visibility

Usage: Public modifier

public ese main() { ... }
public let API_KEY = "secret";

async

Meaning: Asynchronous execution

Usage: Async function modifier

async ese fetch_data() { ... }
async fn process() { ... }

Control Flow

bi / if

Meaning: If / conditional

Usage: Conditional statement

bi x > 0 { ... }
if condition { ... }

bi kosi / else

Meaning: If not / otherwise

Usage: Else clause

bi x > 0 { ... } bi kosi { ... }
if condition { ... } else { ... }

nigita / while

Meaning: While / during

Usage: While loop

nigita x < 10 { ... }
while condition { ... }

fun / for

Meaning: To do / for each

Usage: For loop

fun i ninu items { ... }
for item in collection { ... }

ninu / in

Meaning: Inside / in

Usage: For loop iterator

fun i ninu 0..10 { ... }
for i in range { ... }

match

Meaning: Pattern matching

Usage: Match statement

match (value) {
    0 => Irosu.fo("zero"),
    _ => Irosu.fo("other")
}

Error Handling

gbiyanju / try

Meaning: To attempt / try

Usage: Try block

gbiyanju { ... } gba (e) { ... }
try { ... } catch (e) { ... }

gba / catch

Meaning: To receive / catch

Usage: Catch block

gbiyanju { ... } gba (error) { ... }
try { ... } catch (error) { ... }

nipari / finally

Meaning: In conclusion / finally

Usage: Finally block

gbiyanju { ... } nipari { ... }
try { ... } finally { ... }

ewo / assert

Meaning: Prohibition / assertion

Usage: Assertion statement

ewo x >= 0;
assert condition;

verify

Meaning: To verify

Usage: Assertion alternative

verify balance >= 0;

Imports and Modules

iba / import

Meaning: To borrow / import

Usage: Import statement

iba std.irosu;
import std.utils;

lati / from

Meaning: From / source

Usage: Named import

iba { fo, ka } from std.irosu;
import { print, read } from std.irosu;

Usage Guidelines

Bilingual Support

  • Mixed Usage: Can mix Yoruba and English in same program
  • Consistency: Choose one style per project for consistency
  • Cultural Context: Yoruba keywords preserve cultural meaning
  • Accessibility: English keywords provide accessibility

Best Practices

  • Domain Names: Use Odù domains without imports
  • Type Names: Use type hints for clarity
  • Constants: Use const/loruko for immutable values
  • Error Handling: Use gbiyanju/gba for robust error handling

Reserved Word Restrictions

  • No Shadowing: Cannot shadow reserved words
  • No Redefinition: Cannot redefine reserved words
  • Context Sensitive: Some words are context-sensitive (e.g., public)
  • Future Reserved: Additional words may be reserved in future versions
