Reserved Words & Domain Aliases

    Canonical reference for all reserved keywords and domain names in Ifá-Lang, extracted from crates/ifa-core/src/lexer.rs.

    Reserved Keywords

    These words cannot be used as identifiers. All keywords have Yoruba and English variants.

    Yoruba English Purpose
    ayanmo, àyànmọ́ let Variable declaration
    — const Constant declaration
    ti, bí if Conditional
    bibẹkọ, bíbẹ́kọ́ else Else branch
    fun for For loop
    nigba while While loop
    pada return Return from function
    da break Break loop
    — continue Continue loop
    yàn, yán match Pattern matching
    ese, ẹsẹ fn, function, def Function definition
    odu, ọdù class Class/struct definition
    iba, ìbà import Import module
    otito true Boolean true
    iro false Boolean false
    ohunkohun nil, null Null value
    gbangba pub, public Public visibility
    ikoko, àdáni private Private visibility

    CEN Model Keywords

    Cultural-Enhanced-Naming keywords for resource management and safety.

    Yoruba English Purpose
    ebo, ẹbọ sacrifice RAII resource cleanup block
    ewo, ẹ̀wọ̀ assert, verify Runtime assertion
    ewọ, èèwọ̀ taboo Taboo/forbidden operation marker
    ajose, àjọṣe co-op Reactive binding
    ase, àṣẹ end Block terminator

    Soft Reserved Words

    These identifiers have special meaning in certain contexts but can be used as variable names:

    • self — Current instance in methods
    • super — Parent class reference
    • _ — Wildcard/ignored binding
