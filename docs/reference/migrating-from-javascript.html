Migrating from JavaScript

JavaScript Developer's Guide to Ifá-Lang

Overview

Ifá-Lang shares some concepts with JavaScript but introduces cultural philosophy, protocol-oriented design, and the 16 Odù domain system. This guide helps JavaScript developers transition smoothly.

Key Differences

JavaScript

Object-oriented with prototypes

Ifá-Lang

Protocol-oriented with Odù domains

JavaScript

Dynamic typing with duck typing

Ifá-Lang

Dynamic typing with structural subtyping

JavaScript

Global functions and objects

Ifá-Lang

16 Odù domains (built-in, no imports)

JavaScript

try/catch/finally error handling

Ifá-Lang

gbiyanju/gba/nipari with cultural proverbs

Basic Syntax Comparison

Variables and Functions

JavaScript 
// Variables
let x = 5;
const name = "Alice";

// Functions
function add(a, b) {
    return a + b;
}

// Arrow functions
const multiply = (a, b) => a * b;
Ifá-Lang 
// Variables
ayanmo x = 5;
const name = "Alice";

// Functions
ese add(a, b) {
    pada a + b;
}

// No arrow functions - use regular functions
ese multiply(a, b) {
    pada a * b;
}

Control Flow

JavaScript 
// If statement
if (x > 0) {
    console.log("Positive");
} else {
    console.log("Non-positive");
}

// For loop
for (let i = 0; i < 10; i++) {
    console.log(i);
}

// While loop
while (x < 10) {
    x++;
}
Ifá-Lang 
// If statement
bi x > 0 {
    Irosu.fo("Positive");
} bi kosi {
    Irosu.fo("Non-positive");
}

// For loop
fun i ninu 0..10 {
    Irosu.fo(i);
}

// While loop
nigita x < 10 {
    x = x + 1;
}

Error Handling

JavaScript 
try {
    riskyOperation();
} catch (error) {
    console.error("Error:", error.message);
} finally {
    cleanup();
}
Ifá-Lang 
gbiyanju {
    risky_operation();
} gba (error) {
    Irosu.fo("Error: " + error.message);
} nipari {
    cleanup();
}

Data Structures

Arrays and Objects

JavaScript 
// Arrays
const numbers = [1, 2, 3, 4, 5];
const mixed = [1, "hello", true];

// Objects
const person = {
    name: "Alice",
    age: 30,
    active: true
};

// Array methods
numbers.push(6);
numbers.forEach(x => console.log(x));
const doubled = numbers.map(x => x * 2);
Ifá-Lang 
// Lists
ayanmo numbers = [1, 2, 3, 4, 5];
ayanmo mixed = [1, "hello", otito];

// Maps
ayanmo person = {
    "name": "Alice",
    "age": 30,
    "active": otito
};

// List methods (Ogunda domain)
numbers.push(6);
Ogunda.forEach(numbers, |x| Irosu.fo(x));
ayanmo doubled = Ogunda.map(numbers, |x| x * 2);

String Operations

JavaScript

Ifá-Lang 
ayanmo text = "Hello, World!";
ayanmo upper = Ika.tikun(text);
ayanmo parts = Ika.pin(text, ",");
ayanmo joined = Ika.papo(parts, " ");
ayanmo length = Ika.length(text);

Advanced Concepts

Classes vs Odù Domains

JavaScript 
class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    
    greet() {
        return `Hello, I'm ${this.name}`;
    }
}

const alice = new Person("Alice", 30);
console.log(alice.greet());
Ifá-Lang 
odu Person {
    ayanmo name: Str;
    ayanmo age: Int;
    
    ese greet() -> Str {
        pada "Hello, I'm " + this.name;
    }
}

ayanmo alice = Person();
alice.name = "Alice";
alice.age = 30;
Irosu.fo(alice.greet());

Async/Await

JavaScript 
async function fetchData() {
    try {
        const response = await fetch('/api/data');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error:', error);
    }
}

fetchData().then(data => console.log(data));
Ifá-Lang 
async ese fetch_data() {
    gbiyanju {
        ayanmo response = await Otura.beere('/api/data');
        ayanmo data = await Otura.json(response);
        pada data;
    } gba (error) {
        Irosu.fo("Error: " + error.message);
    }
}

fetch_data().then(|data| Irosu.fo(data));

iba { add, PI } from math;

Unique Ifá-Lang Concepts

Resource Management ( )

Ifá-Lang has automatic resource management that doesn't exist in JavaScript:

Resource Management 
// JavaScript - manual cleanup
const file = await fs.open('data.txt');
try {
    await file.write('content');
} finally {
    await file.close();
}

// Ifá-Lang - automatic cleanup
ebo file = Odi.si("data.txt");
file.write("content");
// File automatically closed when scope ends

Balance Checking (Ìwà Pélé)

Compile-time checking for resource balance:

Balance Checking 
#[iwa_pele]
fn network_task() {
    let conn = Otura.so("example.com", 80);  // open
    // ... work with connection
    conn.pa();  // close - REQUIRED or compile error
}

// Missing conn.pa() would cause compile error!

Reactive Programming (Àjõsé)

Built-in reactive system:

Reactive Binding 
// Create reactive binding
ajose!(counter.value => label.text);

// When counter changes, label updates automatically
counter.value = 42;  // label.text becomes "42"

// No manual event listeners needed!

Migration Strategy

Step 1: Learn the Basics

  • Start with variable declarations (ayanmo/let)
  • Learn basic control flow (bi/if, nigita/while)
  • Practice with Irosu domain for I/O
  • Get comfortable with bilingual keywords

Step 2: Understand Domains

  • Learn the 16 Odù domains and their purposes
  • Replace built-in functions with domain methods
  • Practice domain-specific operations
  • Understand that domains are built-in (no imports)

Step 3: Advanced Concepts

  • Learn resource management ( )
  • Practice reactive programming (Àjõsé)
  • Understand balance checking (Ìwà Pélé)
  • Explore protocol-oriented design

Step 4: Cultural Integration

  • Learn the philosophical meanings behind concepts
  • Understand Yoruba proverbs in error messages
  • Appreciate the cultural significance of Odù domains
  • Embrace the bilingual nature of the language

Common Pitfalls

Importing Domains

Wrong 
// WRONG - Don't import domains
import std.irosu;
Irosu.fo("Hello");
Correct 
// CORRECT - Use domains directly
Irosu.fo("Hello");

Object Access

JavaScript Style 
// JavaScript style - WRONG in Ifá-Lang
person.name = "Alice";
console.log(person.name);
Ifá-Lang Style 
// Ifá-Lang style
person.name = "Alice";
Irosu.fo(person.name);

Error Handling

JavaScript Style 
// JavaScript style - WRONG
try {
    // code
} catch (e) {
    console.log(e.message);
}
Ifá-Lang Style 
// Ifá-Lang style
gbiyanju {
    // code
} gba (error) {
    Irosu.fo(error.message);
}
