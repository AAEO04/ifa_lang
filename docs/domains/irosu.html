0011 Irosu

Console Domain — Input/Output & Terminal Control

Domain Overview

Irosu (0011) governs console input/output operations — the fundamental interface between programs and users. As the speaker domain, Irosu handles all text-based communication, formatted output, and terminal interaction.

Core Methods

Console Operations 
# Basic output
Irosu.fo(value)              # Print with newline
Irosu.fo_inline(value)       # Print without newline

# Input operations
Irosu.ka(prompt)             # Read line with prompt
Irosu.ka_airi(prompt)        # Read password (no echo)

# Colored output
Irosu.awọ(text, "pupa")     # Red text
Irosu.awọ(text, "ewe")      # Green text
Irosu.awọ(text, "ofeefee") # Yellow text
Irosu.awọ(text, "buluu")    # Blue text
Irosu.awọ(text, "funfun")   # White text

Color Support

Irosu provides built-in color support using Yoruba color names:

  • "pupa" — Red
  • "ewe" — Green
  • "ofeefee" — Yellow
  • "buluu" — Blue
  • "funfun" — White

Binary Pattern

Irosu's binary pattern is 0011, representing focused communication. In the 256 Odù lattice vision, Irosu combinations represent:

  • Irosu + Ogbe (0011 + 1111): System communication — logging, diagnostics
  • Irosu + Odi (0011 + 0010): Error communication — error messages, debugging output
  • Irosu + Ika (0011 + 1001): Text communication — formatted strings, localization

Protocol Implementation

Values implement the Irosu protocol if they can be displayed or input through the console:

  • Display capability: Can be converted to string representation
  • Input parsing: Can be constructed from user input
  • Formatting: Support color and style annotations
