Pseudo-domains

Extended domains that provide additional functionality.

Opele / Oracle - Divination

Method Description Example
Opele.cast() Cast 2-level compound Odù ayanmo odu = Opele.cast();
Opele.cast_compound(n) Cast n-level compound (2-5) Opele.cast_compound(3);
Opele.lineage(compound) Get genealogical description Opele.lineage(gpc);
Opele.depth(compound) Get depth of compound Opele.depth(gpc);
Opele.divine(question) Get divination guidance Opele.divine("What path?");

Statement Syntax

Variable Declaration

// Yoruba
ayanmo name = value;
ayanmo x = 42;
ayanmo message = "Hello";

// English
let name = value;
const PI = 3.14159;

Function Definition

// Yoruba
ese function_name(param1, param2) {
    // body
    pada result;  // return
}

// English
fn add(a, b) {
    return a + b;
}

Conditionals

// Yoruba
ti condition {
    // if true
} bibẹkọ {
    // if false
}

// English
if x > 10 {
    Irosu.fo("Large");
} else {
    Irosu.fo("Small");
}

Loops

// While loop
nigba condition {
    // body
}

// For loop
fun i ninu 1..10 {
    Irosu.fo(i);
}

Assertions

// Runtime assertions
ewo balance > 0;
assert amount >= 0, "Amount must be positive";

Domain Calls

// Domain.method(args)
Irosu.fo("Hello!");
Obara.fikun(5, 3);
Opele.cast_compound(3);
