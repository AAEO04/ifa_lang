Domain Stacks

Pre-built libraries for common application domains

v1.2.0

Domain Stacks are pre-built libraries that extend Ifá-Lang with specialized functionality. They combine multiple Odù domains to provide complete solutions for specific application types.

Available Stacks

Crypto Implemented

Cryptographic operations for secure applications.

ifa-std/stacks/crypto

  • Crypto.sha256(data) - SHA-256 hash
  • Crypto.hmac(key, data) - HMAC signature
  • Crypto.random_bytes(n) - Secure random bytes
  • Crypto.base64_encode(data) - Base64 encoding
  • Crypto.base64_decode(str) - Base64 decoding
  • Crypto.aes_encrypt() - Coming in v1.3

Backend Partial

HTTP servers, routing, and middleware.

ifa-std/stacks/backend

  • Otura.so(host, port) - TCP connect
  • Otura.gba(url) - HTTP GET
  • Otura.ran(url, data) - HTTP POST
  • Backend.route() - Coming in v1.4
  • Backend.middleware() - Coming in v1.4
  • Backend.websocket() - Coming in v1.4

Frontend Planned

HTML/CSS generation and components.

ifa-std/stacks/frontend

  • Frontend.element(tag) - DOM element creation
  • Frontend.style(css) - CSS styling
  • Frontend.render(component) - Render to HTML
  • Frontend.event(el, type, fn) - Event handling

Target: v2.0 (Post-1.0 stable)

GameDev Planned

Game development: sprites, physics, game loop.

ifa-std/stacks/gamedev

  • Ose.nu() - Clear canvas
  • Ose.ya(x, y, char) - Draw point
  • Ose.ila(x1,y1,x2,y2,c) - Draw line
  • Ose.han() - Render frame
  • GameDev.sprite() - Sprite loading
  • GameDev.collision() - Collision detection
  • GameDev.physics() - 2D physics

ASCII canvas available now; full game engine in v2.0

IoT Partial

Embedded systems, sensors, GPIO.

ifa-std/stacks/iot

  • ifa flash --target <board> - Flash firmware
  • Opon.size() - Memory sizing for embedded
  • ifa-embedded - no_std crate
  • IoT.gpio(pin) - GPIO control
  • IoT.sensor(type) - Sensor reading
  • IoT.mqtt() - MQTT protocol

Basic embedded support available; full IoT stack in v1.5

ML Planned

Machine learning and neural networks.

ifa-std/stacks/ml

  • ML.tensor(shape) - Tensor creation
  • ML.layer(type, nodes) - Network layers
  • ML.train(model, data) - Model training
  • ML.predict(model, input) - Inference

Target: v3.0 (Long-term roadmap)

Using Stacks

Import a stack using the iba (import) keyword:

// Import the crypto stack
iba Crypto;

// Use cryptographic functions
ayanmo hash = Crypto.sha256("Hello, World!");
Irosu.fo(hash);

ase;
