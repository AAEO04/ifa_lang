?? Crypto Stack

Production-grade cryptography using ring, sha2, argon2

Features

Hashing

SHA-256, SHA-512, HMAC-SHA256

// SHA-256 hash
ayanmo hash = Crypto.sha256("message");

// HMAC for authentication
ayanmo mac = Crypto.hmac("key", "message");

Password Hashing

Argon2id for secure password storage

// Hash password (returns salt+hash)
ayanmo stored = Crypto.hash_password("user_password");

// Verify password
ayanmo valid = Crypto.verify_password("user_password", stored);

Encryption

AES-256-GCM authenticated encryption

// Generate key
ayanmo key = Crypto.random_bytes(32);

// Encrypt
ayanmo encrypted = Crypto.encrypt(key, "secret data");

// Decrypt
ayanmo decrypted = Crypto.decrypt(key, encrypted);

Random

Cryptographically secure random generation

// Random bytes
ayanmo bytes = Crypto.random_bytes(16);

// UUID v4
ayanmo id = Crypto.uuid();

Encoding

Base64, Hex encoding/decoding

ayanmo encoded = Crypto.base64_encode("hello");
ayanmo decoded = Crypto.base64_decode(encoded);
