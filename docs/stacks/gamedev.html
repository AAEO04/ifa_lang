?? GameDev Stack

2D/3D game development primitives

Features

Vectors

Vec2, Vec3 for positions and velocities

ayanmo pos = Game.vec2(100, 200);
ayanmo vel = Game.vec3(0, -9.8, 0);

// Operations
ayanmo moved = Game.add(pos, vel);
ayanmo dist = Game.distance(a, b);

Collision

AABB, Circle collision detection

// Bounding boxes
ayanmo box_a = Game.aabb(0, 0, 50, 50);
ayanmo box_b = Game.aabb(25, 25, 50, 50);

ayanmo hit = Game.collides(box_a, box_b);

// Circle collision
ayanmo circle = Game.circle(100, 100, 25);
ayanmo overlap = Game.circle_collides(circle, box_a);

Sprites

Sprite loading and animation

ayanmo sprite = Game.sprite("player.png");
ayanmo frames = Game.spritesheet("walk.png", 4, 1);

// Animation
ayanmo anim = Game.animation(frames, 100);  // 100ms per frame
Game.play(anim);

ECS

Entity-Component-System architecture

// Create entity
ayanmo player = Game.entity();

// Add components
Game.add_component(player, "position", Game.vec2(0, 0));
Game.add_component(player, "velocity", Game.vec2(1, 0));

// Query entities
ayanmo movers = Game.query(["position", "velocity"]);
