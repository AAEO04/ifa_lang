?? Embedded Runtime

no_std Ifá-Lang for microcontrollers and embedded systems

The ifa-embedded crate provides a minimal, no_std runtime for running Ifá-Lang on embedded systems with limited resources.

Source: crates/ifa-embedded/src/lib.rs (663 lines)

Features

  • no_std compatible — Works without Rust's standard library
  • HAL traits — Generic GPIO, Serial, Timer interfaces
  • Minimal opcodes — Subset of bytecode for small footprint
  • 32-bit values — i32/f32 instead of i64/f64
  • Fixed-size stack — Deterministic memory usage

Configuration Presets

Preset RAM Stack Memory Use Case
minimal() <2KB 16 32 slots ATtiny, very small MCUs
standard() 8-32KB 64 256 slots Arduino, STM32, ESP8266
extended() 64KB+ 256 1024 slots ESP32, Raspberry Pi Pico

HAL Traits

InputPin

trait InputPin

Read-only GPIO input.

trait InputPin {
    fn is_low(&self) -> bool;
    fn is_high(&self) -> bool { !self.is_low() }
}

OutputPin

trait OutputPin

Writable GPIO output.

trait OutputPin {
    fn set_low(&mut self);
    fn set_high(&mut self);
    fn toggle(&mut self);
}

Serial

trait Serial

UART/Serial communication.

trait Serial {
    fn read_byte(&mut self) -> EmbeddedResult;
    fn write_byte(&mut self, byte: u8) -> EmbeddedResult<()>;
    fn write_bytes(&mut self, data: &[u8]) -> EmbeddedResult<()>;
}

DelayUs

trait DelayUs

Timer/delay functionality.

trait DelayUs {
    fn delay_us(&mut self, us: u32);
    fn delay_ms(&mut self, ms: u32) { 
        self.delay_us(ms * 1000); 
    }
}

EmbeddedValue

Simplified value type for embedded contexts (smaller than IfaValue):

enum EmbeddedValue {
    Null,
    Bool(bool),
    Int(i32),    // 32-bit instead of 64-bit
    Float(f32),  // 32-bit instead of 64-bit
}

Usage Example

// Blink LED on GPIO pin 13
ayanmo led = GPIO.output(13);
ayanmo delay = Timer.new();

nigba (otito) {
    led.set_high();
    delay.delay_ms(500);
    led.set_low();
    delay.delay_ms(500);
}

Supported Opcodes

The embedded VM uses a subset of opcodes for minimal footprint:

Nop, LoadConst, LoadTrue, LoadFalse, LoadNull,
Add, Sub, Mul, Div, Mod, Neg,
Eq, Ne, Lt, Le, Gt, Ge,
And, Or, Not,
LoadLocal, StoreLocal,
Jump, JumpIfFalse,
Halt
