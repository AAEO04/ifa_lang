?? FFI - Foreign Function Interface

Call C, Rust, Python, and JavaScript from Ifá-Lang

Overview

The FFI module provides a polyglot bridge for calling external libraries and scripts from Ifá-Lang. It supports three backends:

Native (C/Rust) Python JavaScript

Native Libraries (C/Rust)

ffi.load(name, path)

Load a shared library (.dll, .so, .dylib)

// Load a native library
ffi.load("math", "./libmath.so");

// Bind a function signature
ffi.bind("math", "add", ["i32", "i32"], "i32");

// Call the function
ayanmo result = ffi.call("math", "add", [5, 3]);
Irosu.fo(result);  // 8

Python Integration

ffi.itumo("python", config)

Initialize the Python bridge. Requires Python 3.7+.

// Initialize Python bridge
ffi.itumo("python", {
    module: "numpy",
    interpreter: "/usr/bin/python3"
});

// Call Python functions
ayanmo arr = ffi.call("numpy", "array", [[1, 2, 3]]);
ayanmo mean = ffi.call("numpy", "mean", [arr]);
Irosu.fo("Mean: " + mean);

JavaScript Integration

ffi.itumo("js", config)

Run JavaScript code via the embedded Boa engine.

// Load JS module
ffi.load_js("utils", `
    function greet(name) {
        return "Hello, " + name + "!";
    }
`);

// Call JS function
ayanmo msg = ffi.call("utils", "greet", ["Ifá"]);
Irosu.fo(msg);  // "Hello, Ifá!"
