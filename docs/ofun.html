The Elder - Capabilities & Permissions

🔒 Security Authority: Ọfún controls ALL permissions in Ifá-Lang. By default, scripts run with minimal privileges. Use CLI flags to grant capabilities.

Capability Types

📁 ReadFiles

--allow-read=PATH

Read files under the specified path.

Default: Denied (except script dir) 
Odi.ka("./data/config.json")

✏️ WriteFiles

--allow-write=PATH

Write/create files under the specified path.

Default: Denied 
Odi.ko("output.txt", data)

🌐 Network

--allow-net=DOMAIN

HTTP/TCP connections to specified domains.

Default: Denied 
Otura.gba("https://api.com")

🔑 Environment

--allow-env=KEY

Access environment variables by key.

Default: Denied 
Ogbe.ayika("HOME")

⏰ Time

--allow-time

Access system time and date functions.

Default: Denied 
Iwori.akoko()

🎲 Random

--allow-random

Generate random numbers (CSPRNG).

Default: Allowed 
Owonrin.pese()

💻 Stdio

--allow-stdio

Console input/output operations.

Default: Allowed 
Irosu.fo("Hello!")

⚡ Execute

--allow-run=PROGRAM

Spawn external processes.

Default: Denied 
Ogunda.si("ls -la")

CLI Usage

# Run with specific permissions
ifa run script.ifa --allow-read=./data --allow-net=api.example.com

# Multiple permissions of same type
ifa run script.ifa --allow-env=HOME --allow-env=PATH --allow-env=USER

# All permissions (⚠️ use with caution!)
ifa run script.ifa --allow-all

Rust API

use ifa_sandbox::{CapabilitySet, Ofun};
use std::path::PathBuf;

let mut caps = CapabilitySet::new();

// Grant file read
caps.grant(Ofun::ReadFiles { root: PathBuf::from("./data") });

// Grant network
caps.grant(Ofun::Network { domains: vec!["api.example.com".into()] });

// Check permission
if caps.check(&Ofun::ReadFiles { root: PathBuf::from("./data/file.txt") }) {
    // Access granted
}

Legacy Methods

These methods remain for backwards compatibility but are superseded by the capability system:

ase()

Request elevated permissions (legacy)

fun(permission, granted)

Grant/revoke permission (legacy)

iru(value)

Get type of value (typeof)

oruko()

Get current user/hostname
