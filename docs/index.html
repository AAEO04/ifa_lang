Language Syntax

Variables

Declaration
ayanmọ x = 50; (Yoruba) or let x = 50; (English)

Control Flow

If/Else

ti x > 5 { ... } bibẹkọ { ... }

or: if/else

While Loop

nigba x < 10 { ... }

or: while

For Loop

fun item ninu list { ... }

or: for...in

Match

yàn (x) { 1 => ... }

or: match/select

Functions & Classes

Function (Ese/Verse)
ese greet(name) {
    Irosu.fo("Hello " + name);
}
Class (Odù/Domain)
odù Calculator {
    ayanmọ value = 0;
    ese add(n) {
        value = value + n;
    }
}

Ìwà-Pẹ̀lẹ́ (Balance Rules)

Every resource opened must be closed. The Ìwà Engine enforces good character:

Files

Odi.si() → Odi.pa()

Open → Close

Network

Otura.ether_de() → Otura.ether_pa()

Join → Leave

Locks

Osa.khoa() → Osa.si()

Lock → Unlock

Arrays

Ogunda.ge() → Irete.tu()

Create → Release
