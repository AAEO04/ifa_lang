The Light

The Darkness

Process termination, endings, sleep

The Mirror

Reflection, time, iteration, loops

1001 Òdí (File)

The Vessel

Storage, file operations, containment

The King

The Troublemaker

The Crusher

The Beautifier

Graphics, display, aesthetics

0101 Òfún (Root)

The Creator

Object creation, inheritance

Language Syntax

Variables

Declaration
ayanmọ x = 50; (Yoruba) or let x = 50; (English)

Control Flow

If/Else

ti x > 5 { ... } bibẹkọ { ... }

or: if/else

While Loop

nigba x < 10 { ... }

or: while

For Loop

fun item ninu list { ... }

or: for...in

Match

yàn (x) { 1 => ... }

or: match/select

Functions & Classes

Function (Ese/Verse)
ese greet(name) {
    Irosu.fo("Hello " + name);
}
Class (Odù/Domain)
odù Calculator {
    ayanmọ value = 0;
    ese add(n) {
        value = value + n;
    }
}
