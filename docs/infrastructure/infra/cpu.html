CpuContext

Global CPU context for parallel operations.

CpuContext.configure(threads: usize) ? Result<(), String>

Configure the global thread pool size.

// Set to 4 threads
CPU.configure(4);

// Use all available cores
CPU.configure(CPU.num_threads());

CpuContext.num_threads() ? usize

Get number of available CPU threads.

ayanmo cores = CPU.num_threads();
Irosu.fo("Available cores: " + cores);

Parallel Iterators

CpuContext.par_map(data, map_fn) ? Vec

Parallel map over a slice.

ayanmo numbers = [1, 2, 3, 4, 5, 6, 7, 8];
ayanmo squared = CPU.par_map(numbers, |x| x * x);
// [1, 4, 9, 16, 25, 36, 49, 64]

CpuContext.par_reduce(data, identity, map_fn, combine_fn) ? T

Parallel reduce with identity and combine function.

ayanmo numbers = [1, 2, 3, 4, 5];
ayanmo sum = CPU.par_reduce(numbers, 0, |x| x, |a, b| a + b);
// 15

CpuContext.par_filter(data, predicate) ? Vec

Parallel filter.

ayanmo numbers = [1, 2, 3, 4, 5, 6, 7, 8];
ayanmo evens = CPU.par_filter(numbers, |x| x % 2 == 0);
// [2, 4, 6, 8]

CpuContext.par_sort(data)

Parallel stable sort (in-place).

ayanmo data = [5, 2, 8, 1, 9];
CPU.par_sort(data);
// [1, 2, 5, 8, 9]

CpuContext.par_any(data, predicate) ? bool

Returns true if any element satisfies predicate.

CpuContext.par_all(data, predicate) ? bool

Returns true if all elements satisfy predicate.

CpuContext.par_find(data, predicate) ? Option<T>

Returns first element satisfying predicate.

Task Graph

Async task graph with dependencies for complex workflows.

// Create task graph
ayanmo graph = TaskGraph.new();

// Add tasks with dependencies
ayanmo task_a = graph.add("fetch_data", fetch_data_fn);
ayanmo task_b = graph.add("process_data", process_fn);
ayanmo task_c = graph.add("save_results", save_fn);

// Define dependencies
graph.depends(task_b, task_a);  // B depends on A
graph.depends(task_c, task_b);  // C depends on B

// Execute all tasks
graph.execute();
