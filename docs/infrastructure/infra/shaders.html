Available Shaders

MATMUL_SHADER

Tiled matrix multiplication with shared memory optimization.

  • Tile size: 16×16
  • Workgroup size: 16×16
  • Features: Shared memory tiling, boundary checks
// Usage via GpuContext
ayanmo result = GPU.matmul(buf_a, buf_b, m, n, k);

REDUCE_SUM_SHADER

Tree-based parallel reduction for efficient summation.

  • Workgroup size: 256
  • Features: Shared memory, tree reduction

REDUCE_MAX_SHADER

Parallel max reduction.

SCALE_BIAS_SHADER

Elementwise: output[i] = input[i] * scale + bias

// Normalize data on GPU
ayanmo normalized = GPU.map_scale_bias(input, count, 1/std, -mean/std);

RELU_SHADER

ReLU activation: max(0, x)

// In-place ReLU activation
GPU.relu(buffer, count);

SOFTMAX_SHADER

Numerically stable softmax with max subtraction.

  • Features: Shared memory for max/sum, numerical stability

VEC_ADD_SHADER

Vector addition: c[i] = a[i] + b[i]

ayanmo result = GPU.vec_add(buf_a, buf_b, count);

DOT_PRODUCT_SHADER

Parallel dot product with partial sums.

Custom Shaders

Create your own compute pipelines:

ayanmo my_shader = "
@group(0) @binding(0) var input: array;
@group(0) @binding(1) var output: array;

@compute @workgroup_size(256)
fn main(@builtin(global_invocation_id) id: vec3) {
    let i = id.x;
    output[i] = input[i] * input[i];  // Square each element
}
";

ayanmo pipeline = GPU.get_or_create_pipeline("square", my_shader, "main");
